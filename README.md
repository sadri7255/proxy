## لینک‌های پروکسی:

### 📱 شبکه‌های اجتماعی
- [تلگرام (Google Translate)](https://translate.google.com/translate?sl=auto&tl=en&u=https://web.telegram.org/k/) - باز کردن تلگرام با گوگل ترنسلیت
- [تلگرام (Google Cache)](https://webcache.googleusercontent.com/search?q=cache:web.telegram.org/k/) - نسخه کش شده
- [اینستاگرام](https://translate.google.com/translate?sl=auto&tl=en&u=https://www.instagram.com/) - از طریق ترنسلیت
- [توییتر موبایل](https://translate.google.com/translate?sl=auto&tl=en&u=https://mobile.twitter.com/) - نسخه موبایل
- [ردیت نسخه قدیمی](https://translate.google.com/translate?sl=auto&tl=en&u=https://old.reddit.com/) - old.reddit.com

### 🔍 موتورهای جستجو بعنوان پروکسی
- [Google Web Light](https://googleweblight.com/?lite_url=https://web.telegram.org/k/) - نسخه سبک گوگل
- [Bing](https://www.bing.com/search?q=web.telegram.org) - جستجو در بینگ
- [Baidu](https://www.baidu.com/s?wd=web.telegram.org) - جستجو در بایدو

- [Hide.me Proxy](https://hide.me/en/proxy) - پروکسی Hide.me
- [4EverProxy](https://www.4everproxy.com/) - پروکسی 4Ever
